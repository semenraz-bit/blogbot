                                Комплект Методических Рекомендаций По Разработке Инструкций По Охране Труда!

        Приветствую, друзья! В помощь руководителям и специалистам, в чьи должностные обязанности входит разработка инструкций по охране труда, выкладываю комплект Методических рекомендаций по разработке инструкций по охране труда, утверждённых в своё время Минтрудом России. Предполагаю, что какая-то часть подписчиков удивилась на слове комплект. Всё так — комплект из 14 Методических рекомендаций по охране труда для разных отраслей экономики и видов работ. Возможно для кого-то из подписчиков это будет открытием 🙂

Методические рекомендации по охране труда — разрабатываем инструкции по Фэншуй
Итак, в комплект вошли:
1. Методические рекомендации по разработке инструкций по охране труда (утв. Минтрудом РФ 13.05.2004 г.)
2. Методические рекомендации по разработке инструкций по охране труда при выполнении сварочных и газорезательных работ (утв. Минтрудом РФ 14.05.2004 г.)
3. Методические рекомендации по разработке инструкций по охране труда при выполнении работ с ручным инструментом и приспособлениями (утв. Минтрудом РФ 07.05.2004 г.)
4. Методические рекомендации по разработке инструкций по охране труда для работников оптовых баз, складов, рынков, холодильников (утв. Минтрудом РФ 19.05.2004 г.)
5. Методические рекомендации по разработке инструкций по охране труда для работников, занятых эксплуатацией газового хозяйства организаций (утв. Минтрудом РФ 21.05.2004 г.)
6. Методические рекомендации по разработке инструкций по охране труда для работников стационарных и передвижных автозаправочных станций, нефтебаз, складов ГСМ (утв. Минтрудом России 17.05.2004 г.)
7. Методические рекомендации по разработке инструкций по охране труда для работников, занятых бытовым обслуживанием населения (утв. Минтрудом России 18.05.2004 г.)
8. Методические рекомендации по разработке инструкций по охране труда для основных профессий и видов работ в лесозаготовительной и деревоперерабатывающей промышленности (утв. Минтрудом РФ 11.05.2004 г.)
9. Методические рекомендации по разработке инструкций по охране труда для основных профессий и видов работ в мебельном производстве (утв. Минтрудом России 11.05.2004 г.)
10. Методические рекомендации по разработке инструкций по охране труда для работников, занятых в производстве цемента и утилизации отходов (утв. Минтрудом РФ 20.05.2004 г.)
11. Методические рекомендации по разработке инструкций по охране труда при использовании стеклопластиков, стекловолокна и стеклоткани (утв. Минтрудом РФ 13.05.2004 г.)
12. Методические рекомендации по разработке инструкций по охране труда для работников, занятых обслуживанием и ремонтом фреоновых холодильных установок и оборудования охлаждаемых помещений (утв. Минтрудом РФ 12.05.2004 г.)
13. Методические рекомендации по разработке инструкций по охране труда для работников, занятых в производстве асбестоцементных изделий и утилизации отходов (утв. Минтрудом РФ 20.05.2004 г.)
14. Методические рекомендации по разработке инструкций по охране труда в организациях гражданской авиации (утв. Распоряжением Минтранса РФ от 25.04.2002 г. № НА-141-р)
Также при использовании данных Методических рекомендаций надо учитывать, что некоторые нормативные правовые акты, на которые идут ссылки с Методических рекомендаций, утратили силу.
Для новичков в охране труда напоминание — приведённые здесь Методические рекомендации, не являются нормативными правовыми актами и следовательно не обязательны для применения.
Если нужны готовые инструкции по охране труда, то у нас есть БАЗА ИНСТРУКЦИЙ ПО ОХРАНЕ ТРУДА.
На этом всё. Если материал пригодился, ставим рейтинговые звёздочки и отписываемся в комментариях 😉 Спасибо за участие!
