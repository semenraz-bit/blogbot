Комплект «охрана труда для частных охранных предприятий» содержит следующие документы:
1. СТАНДАРТ об организации работы по охране труда в ЧОП;
2. Программа по проведению вводного инструктажа по охране труда в ЧОП;
3. Инструкция по проведению вводного инструктажа по охране труда в ЧОП;
4. Лекция по охране труда для ЧОП;
5. Программа обучения по охране труда для руководителей и специалистов ЧОП;
6. Программа по проведению первичного инструктажа по охране труда для водителя;
7. Инструкция по проведению первичного инструктажа по охране труда для водителя
8. Программа обучения и экзаменационный тест по охране труда для должности водитель легкового автомобиля;
9. Программа обучения по охране труда и экзаменационный тест для должности – старший объекта, старший смены, водитель — охранник, охранник 4,5,6 разрядов;
10. Программа по проведению первичного инструктажа по охране труда старших объекта, старших смен, водителей – охранников, охранников 4,5,6 разрядов в структурном подразделении;
11. Инструкция по проведению первичного инструктажа по охране труда для старшего объекта, старшего смены, водителя – охранника, охранника 4,5,6 разряда;
12. СПИСОК должностей освобожденных от первичного инструктажа;
13. Инструкция по охране труда для работников Аппарата Управления;
14. Инструкция по охране труда для административно-управленческого персонала, специалистов, инженерно-технического персонала, инженерно-технических работников и младшего обслуживающего персонала;
15. Инструкция по охране труда для начальника отдела (группы) охраны;
16. Инструкция по охране труда для охранника при осуществлении видеоконтроля и видеонаблюдения;
17. Инструкция по охране труда для водителей-охранников;
18. Инструкция по охране труда для водителя;
19. Инструкция по охране труда для охранника КПП;
20. Инструкция по охране труда для охранника офисных помещений;
21. Инструкция по охране труда для старших объекта, старших смен, водителей – охранников, охранников 4,5,6 разрядов;
22. Инструкция по охране труда для охранников;
23. Инструкция по охране труда для охранника группы досмотра железнодорожного транспорта;
24. Инструкция по оказанию первой помощи при несчастных случаях на производстве
