        Приветствую, друзья! Бесплатная программа для организации работы по охране труда и смежным направлениям — информационная система специалиста по охране труда (далее — ИС СОТ), обновилась до версии 5.0! Смотрим нововведения, скачиваем, настраиваем и радуемся автоматизации процессов.

Новые возможности ИС СОТ 5
В новой версии программы вас ждёт:

Проработана структура по направлениям работы специалиста по охране труда.
Доработан учёт инструктажей работников по охране труда и ПБ (в форме ввода данных персонала добавлен раздел, введены логические поля).
Разработана форма по учёту проведения инструктажей, в том числе и по подразделениям. Введён подсчёт оставшихся дней до очередных инструктажей работников.
Разработана форма по учёту СОУТ с графическим представлением данных.
В таблицу по учёту карт СОУТ введено логическое поле (СИЗ) и ведена подстановка из списка в полях «Должности» и «Подразделения» — для удобства ввода карт или их замены, а также при изменении (добавлении) подразделений и должностей.
Разработана форма по медицинскому освидетельствованию персона.
Применён визуальный графический контроль проведения периодического медицинского обследования персонала.
В форму учёта аттестаций по промышленной безопасности добавлено поле для учёта аттестаций по категориям персонала и диаграммы.
Сформированы запросы для Диаграмм и Отчётов по персоналу.
В «Органайзер основных задач (мероприятий)» добавлена ссылка для перехода к записной книжке MS OneNote и введены оперативные Диаграммы.
Введен расчёт возраста и стажа работников в форму ввода данных персонала.
Разработана форма «Персонал — информационная панель».
В контрольный лист введён значок «Медицинский контроль» и поле «Период» медицинского осмотра.
Введена блокировка полей данных от случайных изменений при работе с ИС СОТ в подчинённых формах.
Количество наименований средств индивидуальной защиты в личных карточках учёта выдачи СИЗ работников увеличено до 15.
В формы «Аттестации по промышленной безопасности» и «Аттестации по ГО и ЧС» добавлена кнопка «Удалить запись».
Подготовлена инструкция по порядку работы с формой «Ввода данных персонала организации».
Все формы ИС СОТ оптимизированы под разрешение монитора 1920×1080.
ИС СОТ обеспечивает работу в компьютерной сети предприятия по принципу «Клиент-сервер» (с использованием штатных средств MS Access для совместного использования «разделенной» базы данных).
Учёт проводимых Аттестаций и Инструктажей работников может производиться непосредственно руководителями структурных подразделений в серверную часть ИС СОТ, используя компьютерную сеть.
